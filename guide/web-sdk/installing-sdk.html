# Installing the FunFair Wallet SDK

# NPM

$ npm install @funfair-tech/wallet-sdk

# YARN

$ npm install @funfair-tech/wallet-sdk

# Reference the Web SDK

Once installed you need to drop the below script into your <head> HTML tag:

<script
  src="https://dev.wallet.funfair.io/assets/sdk/fun-wallet-sdk.js"
  type="text/JavaScript"
></script>

# Example

<!DOCTYPE html>
<html lang="en">
  <head>
    <script
      src="https://dev.wallet.funfair.io/assets/sdk/fun-wallet-sdk.js"
      type="text/JavaScript"
    ></script>
  </head>
  <body></body>
</html>

# Peer Dependencies

"post-robot": "9.0.36"

# NPM

$ npm install post-robot@9.0.36

# YARN

$ yarn add post-robot@9.0.36

It should be fine to use the latest version of post-robot; however, if you're only installing it to allow the SDK to work then we recommend installing the exact version that's referenced.
